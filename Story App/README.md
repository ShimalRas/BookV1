# Story App

A local desktop character manager for building RPG-style sheets, tracking stat changes, and reviewing skill history.

## Features
- Character sheets with rank, race, level, stats, divinity, and notes
- Skill browser with rank and description editing
- Change history stored locally in SQLite
- Progression panel for rank gains and tier benchmarks

## Run

This project is scaffolded for Python and Tkinter.

```bash
python main.py
```

## Data

The app stores its database locally in the `data/` folder next to the source files.
